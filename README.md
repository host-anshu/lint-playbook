# Lint Playbook

A simple tool to check [ansible-playbooks](http://docs.ansible.com/ansible/playbooks.html) for 
logical consistencies. It uses [Simple Interceptor](https://github.com/host-anshu/simpleInterceptor) 
to achieve this.
